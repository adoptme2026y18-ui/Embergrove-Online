# Embergrove Online - Game Design Document

## Overview

Embergrove Online is a realistic 3D fantasy RPG focused on exploration, crafting, farming, housing, and social gameplay.

## Core Pillars

1. Relaxing Gameplay
2. Social Interaction
3. Meaningful Progression
4. Player Freedom
5. Living World

## Target Audience

Players who enjoy:
- RPGs
- Crafting
- Exploration
- Farming
- Multiplayer experiences

## Gameplay Loop

Explore → Gather → Craft → Improve Skills → Unlock New Opportunities → Socialize
