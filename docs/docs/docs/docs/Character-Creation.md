# Embergrove Online - Character Creation

## Overview

Players create a unique character before entering the world of Embergrove.

Character creation focuses on appearance and identity rather than gameplay advantages.

## Appearance Options

### Body Type
- Option 1
- Option 2
- Option 3

### Skin Tone
- Multiple choices

### Face
- Multiple face presets

### Eyes
- Eye shape
- Eye color

### Hair
- Hairstyles
- Hair colors

### Facial Features
- Eyebrows
- Facial hair
- Scars
- Freckles

## Voice

Players may choose from several voice options.

## Starting Outfit

Players select a starter outfit style.

Examples:
- Adventurer
- Artisan
- Farmer
- Traveler

Outfit choice does not affect stats.

## Character Name

Players choose a unique character name.

Naming rules:
- Appropriate language only
- Length limits
- No impersonation

## Starting Location

All new players begin in Ashgrove Valley.

## Future Features

- Additional customization options
- Cosmetic unlocks
