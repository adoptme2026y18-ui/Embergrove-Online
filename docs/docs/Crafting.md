# Embergrove Online - Crafting

## Overview

Crafting allows players to turn gathered resources into useful items, equipment, decorations, and consumables.

Players improve crafting skills by creating items.

## Crafting Professions

### Blacksmithing

Creates:
- Swords
- Axes
- Hammers
- Armor
- Tools

Resources:
- Ore
- Ingots
- Coal

### Carpentry

Creates:
- Furniture
- Storage
- Decorations
- Wooden tools

Resources:
- Logs
- Planks

### Tailoring

Creates:
- Clothing
- Light armor
- Bags
- Decorative items

Resources:
- Cloth
- Leather
- Thread

### Cooking

Creates:
- Meals
- Snacks
- Buff foods

Resources:
- Crops
- Fish
- Meat
- Herbs

### Alchemy

Creates:
- Potions
- Tonics
- Magical consumables

Resources:
- Herbs
- Mushrooms
- Magical ingredients

## Crafting Stations

Examples:
- Forge
- Workbench
- Loom
- Kitchen
- Alchemy Table

## Recipe System

Recipes can be obtained through:
- Skill progression
- Quests
- Exploration
- Vendors
- Events

## Item Quality

Possible quality levels:
- Common
- Uncommon
- Rare
- Epic
- Legendary

Higher crafting skill increases the chance of producing higher-quality items.

## Future Features

- Crafting specializations
- Masterwork items
- Player-created markets
- Rare recipe discoveries
