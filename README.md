# Embergrove Online

A relaxing and social 3D fantasy RPG where players can explore, craft, farm, build homes, and adventure with friends.

## Vision

Embergrove Online is designed to be a welcoming fantasy world where players can choose their own path. Whether exploring ancient ruins, running a farm, crafting equipment, or socializing with friends, every playstyle is meaningful.

## Features (Planned)

- Realistic 3D world
- Character customization
- Skill-based progression
- Gathering and crafting
- Farming and animal care
- Player housing
- Multiplayer and co-op gameplay
- Guilds and communities
- Player-driven economy

## Current Status

Early planning and game design phase.

## Development

This repository contains design documents, planning materials, and future game development resources for Embergrove Online.
